--------------------------------------------------------------------------- Binds the given channel to a tuner.

Note that channels may be bound to more than one tuner.

Upon a successful binding, the following actions are taken: o The tuner ID is stored with the channel object o A flowgraph is built (if the channel is not already bound to a tuner) or augmented (if the channel is already bound) in the dataplane for the channel o A PLI-NACK is sent to initiate RCC (if the channel is not already bound to a tuner)

\begin{Desc}
\item[Parameters:]
\begin{description}
\item[\mbox{$\leftarrow$} {\em chanid}]Channel to be bound \item[\mbox{$\leftarrow$} {\em tid}]Tuner to which the channel should be bound (specified by its dataplane tuner ID). \item[\mbox{$\leftarrow$} {\em bp}]Bind parameters \item[\mbox{$\leftarrow$} {\em chan\_\-event\_\-cb,:}]chan event cb function \item[\mbox{$\rightarrow$} {\em vqec\_\-chan\_\-err\_\-t}]Returns VQEC\_\-CHAN\_\-ERR\_\-OK on success, and an error code on failure. --------------------------------------------------------------------------- \end{description}
\end{Desc}
